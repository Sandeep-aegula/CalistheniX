## Overview

The CalistheniX Mission System is a comprehensive progression framework that organizes 27+ calisthenics workouts into four progressive levels, each with specific learning objectives and skill development paths. The system includes:

- **4 Difficulty Levels** (23 missions total)
- **7 Skill Badges** (category-specific achievements)
- **5 Milestone Tiers** (overall progression ranks)
- **Progressive Unlock System** (prerequisites for advanced missions)
- **XP Multiplier Scaling** (increasing rewards for harder content)

## ‚ö° XP & Rewards System

### Base XP by Level
```
Level 1 (Beginner):    20-30 XP (x1.0 multiplier)
Level 2 (Fat Burn):    40-50 XP (x1.2 multiplier = 48-60 actual)
Level 3 (Intermediate): 40-55 XP (x1.5 multiplier = 60-82 actual)
Level 4 (Pro):         75-125 XP (x2.0 multiplier = 150-250 actual)
```

### XP Calculation Formula
```javascript
actualXP = baseXP √ó levelMultiplier
```

### Example XP Gains
- Complete Pull Foundation Tuesday: 25 XP
- Complete Saturday Endurance: 50 √ó 1.2 = 60 XP
- Complete Front Lever: 100 √ó 2.0 = 200 XP
- Unlock Legendary Athlete: +1000 XP bonus

### 3. `useMissionSystem.js` Hook
React hook for managing mission system state and operations.

**Methods:**
```javascript

  userProgress,           // Current user progress data
  loading,                // Loading state
  error,                  // Error messages
  completeMission,        // Function to complete mission
  getAvailableMissions,   // Get unlocked missions
  getLockedMissions,      // Get locked missions with reasons
  getMissionsByLevel,     // Filter by level
  hasSkillBadge,          // Check badge unlock
  hasMilestone,           // Check milestone unlock
  getCurrentTier,         // Get current achievement tier
  getNextMilestoneProgress // Get progress towards next milestone
} = useMissionSystem(userId)
```

**Example:**
```jsx

    // Component JSX
```

## üîå API Endpoints

### POST `/api/missions/complete`
Complete a mission and track progress.

**Request:**
```json
{
  "userId": "user_id",
  "missionId": "mission_id"
}
```

**Response:**
```json
{
  "success": true,
  "xpGained": 60,
  "newTotalXP": 500,
  "totalWorkouts": 10,
  "currentStreak": 5,
  "missionLevel": "fat-burn",
  "newBadges": [...],
  "newMilestones": [...]
}
```

### GET `/api/missions/progress`
Get user's current progress.

**Query Parameters:**
- `userId` (required): User ID

**Response:**
```json
{
  "success": true,
  "progress": {
    "totalWorkouts": 10,
    "totalXP": 500,

    "completedMissions": [...],
    "unlockedSkillBadges": [...],
    "unlockedMilestones": [...],
```

## üéÆ Game Design Philosophy

### Progressive Difficulty Curve
- **Beginner:** Build confidence and consistency
- **Fat Burn:** Increase work capacity
- **Intermediate:** Master advanced techniques
- **Pro:** Achieve elite-level performance

### Motivation Mechanics
1. **Clear Progression:** See exactly what's needed to unlock next mission
2. **Frequent Rewards:** XP for every completion
3. **Multiple Achievement Paths:** Badges and milestones
4. **Streak System:** Encourage daily training
5. **Visible Prerequisites:** Understand the training roadmap

### Accessibility
- Start anywhere with Beginner missions
- Progress at your own pace
- No time limits on missions
- Can repeat missions unlimited times
- Track personal best performances
